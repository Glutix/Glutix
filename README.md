## 📌 Proyectos destacados

Sistema de gestión de gimnasio usado en Iron Gym. Gestión de socios, membresías con alertas de vencimiento y notificaciones automáticas vía WhatsApp.
`Astro` `React` `Node.js` `SQLite` `Tauri`

`Next.js` `TypeScript` `Tailwind CSS` `PostgreSQL` `Express.js` `Prisma`

Marketplace de compra/venta con autenticación, búsquedas avanzadas y panel administrativo.
`React` `TypeScript` `Redux` `Node.js` `PostgreSQL` `Sequelize`
